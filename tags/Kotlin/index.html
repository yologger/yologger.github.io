[Kotlin] 변수(Variable)와 상수(Constant)

Jan 01, 2019 in Kotlin

Kotlin의 특징과 개념에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

이번 포스트에서는 Kotlin의 변수(Variable)와 상수(Constant)에 대해 알아보겠습니다.

변수

변수를 선언할 때는 키워드var를 사용합니다.

변수는 값을 초기화한 후 변경할 수 있습니다.

상수

상수를 선언할 때는 키워드 val을 사용합니다. 상수는 초기화한 후 값을 변경할 수 없습니다.

또한 상수는 선언과 초기화를 동시에 해야합니다.

[Kotlin] Java 유료화 논쟁과 Kotlin

Jan 01, 2019 in Kotlin

Kotlin의 특징과 개념에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

Kotlin

Kotlin은 IntelliJ와 Android Studio를 만든 JetBrains가 2010년에 출시한 프로그래밍 언어입니다. Google은 2017년 안드로이드 공식 언어로 Kotlin으로 추가합니다. Java의 유료화, 그리고 Java API 저작권에 대한 Google과 Oracle의 기나긴 소송의 여파로 보는 시각이 많습니다.

Java에 익숙한 많은 개발자들이 Kotlin이라는 새로운 언어를 배우는데 부담을 느끼고 있습니다. 그러나 Java을 알고있다면 Kotlin도 쉽게 배울 수 있습니다. 앞으로 Kotlin의 특징과 개념에 대해 다루도록 하겠습니다.
