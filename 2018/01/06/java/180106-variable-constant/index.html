[Java] 변수(Variable)와 상수(Constant)

Jan 06, 2018 in Java

Java의 특징과 개념에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

이전 포스트에서는 Java의 자료형(Data Type) 에 대해 알아봤습니다. 이번 포스트에서는 Java의 변수(Variable)와 상수(Constant)에 대해 알아보겠습니다.

변수(Variable)

변수는 자료형을 변수명 앞에 붙여서 선언합니다.

변수는 값을 초기화한 후 변경할 수 있습니다.

상수(Constant)

상수는 자료형 앞에 키워드final를 붙여줍니다. 보통 상수의 이름은 대문자와 언더바(_)를 사용합니다.

상수는 값을 한번 할당하면 변경할 수 없습니다.

마치며

이번 포스트에서는 Java의 변수(Variable)와 상수(Constant)에 대해 알아봤습니다. 다음 포스트에서는 배열(Array)에 대해 알아보겠습니다.
