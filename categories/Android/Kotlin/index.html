Variable and Constant

Dec 21, 2020 in Android, Kotlin

Introduction

In this post, let’s explore how to use variable and constant.

Variable

Use keywork var to declare variable.

You can’t change the value of variable after initialization.

Constant

You can’t change the value of constant after initialization. Use keywork val to declare constant.

You have to declare and initialize at the same time.
