집합 자료형

2018/01/04 카테고리 Kotlin

집합자료형(Collection)은 같은 타입의 데이터를 여러개 담을 수 있습니다. Kotlin은 List, Set, Map이라는 3개의 집합자료형을 지원합니다. List, Set, Map은 모두 Collection클래스를 상속받습니다.

1. List

변경 불가능한 List를 선언할 때는 listOf()메소드를 사용합니다.

변경 가능한 List를 선언할 때는 mutableListOf()메소드를 사용합니다.

2. Set

변경 불가능한 Set를 선언할 때는 setOf()메소드를 사용합니다.

변경 가능한 Set를 선언할 때는 mutableSetOf()메소드를 사용합니다.

3. Map

변경 불가능한 Map을 선언할 때는 mapOf()메소드를 사용합니다.

변경 가능한 Map를 선언할 때는 mutableMapOf()메소드를 사용합니다.

집합자료형 함수

집합자료형은 데이터 조작을 위한 다양한 메소드를 제공합니다. 대표적인 메소드 몇 가지만 알아보겠습니다.
