들어가며

이전 포스트에서는 프래그먼트(Fragment)에 대해 알아봤습니다. 이번 포스트에서는 Application클래스에 대해 알아보겠습니다.

Application 클래스

Application클래스는 앱의 실행과 동시에 생성되는 객체입니다. Application클래스는 메모리에 오직 하나의 인스턴스로만 존재하는 싱글톤 객체며, 주로 컴포넌트 사이에서 데이터를 공유하는데 사용할 수 있습니다.
