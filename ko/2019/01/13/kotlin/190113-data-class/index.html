코틀린(Kotlin)에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

이전 포스트에서는 인터페이스(Interface)에 대해 알아봤습니다. 이번 포스트에서는 데이터 클래스(data class)에 대해 알아보겠습니다.

데이터 클래스(data class)

코틀린은 데이터만을 담기 위한 클래스를 제공합니다. 이땐 키워드 data를 클래스 앞에 붙입니다. 주의할 점은 멤버 변수를 반드시 기본 생성자에 추가해야 합니다.

데이터 클래스는 데이터를 처리하는데 유용한 메소드들을 자동으로 제공합니다. 자주 사용하는 몇가지 메소드만 알아보겠습니다.

(1) equal()

equal()메소드는 두 인스턴스가 동일한지 비교하는데 사용합니다.

(2) copy()

copy()메소드는 인스턴스를 복사하는데 사용합니다.

(3) toString()

toString()메소드는 인스턴스를 문자열로 변환하는데 사용합니다.

마치며

이번 포스트에서는 데이터 클래스(data class)에 대해 알아봤습니다. 다음 포스트에서는 열거 클래스(enum class)에 대해 알아보겠습니다.
