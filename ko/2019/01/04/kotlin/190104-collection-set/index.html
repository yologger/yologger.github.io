[코틀린] 집합자료형(Collection) - (2) Set

2019/01/04 카테고리 Kotlin

코틀린(Kotlin)에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

이전 포스트에서는 집합자료형의 List에 대해 알아봤습니다. 이번 포스트에서는 집합자료형에서 집합(Set)에 대해 알아보겠습니다.

집합(Set)

변경 불가능한 Set를 선언할 때는 setOf()메소드를 사용합니다.

변경 가능한 Set를 선언할 때는 mutableSetOf()메소드를 사용합니다.

Set은 중복을 허락하지 않습니다. 동일한 데이터를 여러 개 넣어도 1개만 유지하고 있습니다.

마치며

이번 포스트에서는 집합(Set)에 대해 알아봤습니다. 다음 포스트에서는 맵(Map)에 대해 알아보겠습니다.
