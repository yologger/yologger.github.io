[Kotlin] 스코프 함수 - (1) run

2019/01/23 카테고리 Kotlin

Kotlin의 특징과 개념에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

코틀린 표준 라이브러리는 객체의 컨텍스트 내에서 코드 블럭을 실행하기 위한 목적만을 가진 5개의 스코프 함수(Scope function)를 제공합니다. run, let, apply, also, with가 그것입니다. 앞으로 5개의 포스트에서 스코프 함수에 대해 알아보려고 합니다.

  1. run() ⬅️ ⬅️ ⬅️
  2. let()
  3. apply()
  4. also()
  5. with()

이번 포스트에서는 run()함수에 대해 알아보겠습니다.

run() 함수

run()함수는 run()함수를 호출한 객체가 코드 블록{}으로 암시적으로 전달됩니다. 따라서 코드 블록에서 호출 객체의 멤버변수와 메소드에 접근할 수 있습니다. 또한 코드 블록의 수행결과가 반환됩니다.

다음처럼 반환값이 없이도 사용할 수도 있습니다.
