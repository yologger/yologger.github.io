[Kotlin] Object, Companion Object 그리고 싱글톤(Singleton)

2019/01/15 카테고리 Kotlin

Kotlin의 특징과 개념에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

3. Companion object

키워드 Companion object는 Java의 static과 유사합니다. Java에서는 멤버변수나 메소드에 키워드static을 붙일 수 있습니다. 이렇게 정의한 static 멤버변수나 static 메소드는 클래스의 인스턴스를 별도로 생성하지 않고 사용할 수 있습니다.
(Java의 static은 이 포스트에서 확인할 수 있습니다.)

static 멤버변수에 할당된 값은 클래스의 모든 인스턴스가 공유합니다.
