객체지향 프로그래밍

Kotlin은 모든 대상을 객체(Object)로 바라봅니다. 이러한 점에서 Kotlin은 객체지향 프로그래밍 언어입니다. 객체지향 프로그래밍을 이해하려면 클래스와 인스턴스에 대해 알아야합니다.

클래스와 인스턴스

클래스와 인스턴스를 설명할 때 와플 기계과 와플을 예로 많이 듭니다.
