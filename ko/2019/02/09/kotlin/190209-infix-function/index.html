[Kotlin] 중위 함수(Infix Function)

2019/02/09 카테고리 Kotlin

Kotlin의 특징과 개념에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

이전 포스트에서는 확장 함수(Extension Function)에 대해 알아봤습니다. 이번 포스트에서는 중위 함수(Infix Function)에 대해 알아보겠습니다.

확장 함수(Extenson Function)

우리는 이전 포스트에서 확장 함수(Extenson Function)에 대해 배웠습니다. 확장 함수를 사용하여 두 정수의 곱을 반환하는 메소드를 Int 클래스에 추가해보겠습니다.

중위 함수(Infix Function)

일반적으로 메소드를 호출할 땐 점(.)과 함수이름, 소괄호를 사용하여 호출합니다. 이때 중위 함수(Infix Function)를 사용하면 점(.)과 소괄호를 생략하고 다음과 같이 호출할 수 있습니다.

중위함수를 사용하려면 3가지 조건을 충족해야합니다.

  1. 중위함수는 클래스의 메소드 또는 확장함수의 형태여야 합니다.
  2. 오직 하나의 인자만 가져야합니다.
  3. 키워드 infix를 붙여줍니다.

마치며

이번 포스트에서는 중위 함수(Infix Function)에 대해 알아봤습니다. 다음 포스트에서는 인라인 함수(Inline Function)에 대해 알아보겠습니다.
