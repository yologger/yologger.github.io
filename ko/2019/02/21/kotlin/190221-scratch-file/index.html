[코틀린] 안드로이드 스튜디오에서 코틀린 연습하기 - (2) Scratch File

2019/02/21 카테고리 Kotlin

코틀린(Kotlin)에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

이전 포스트에서는 REPL에 대해 알아봤습니다. 이번 포스트에서는 안드로이드 스튜디오의 Scratch File에 대해 알아보겠습니다.

Scratch File

스크래치 파일을 사용하면 복잡한 소스 코드도 쉽게 테스트 해볼 수 있습니다.

안드로이드 스튜디오에서 File > New > Scratch File를 선택합니다.

Kotlin을 선택하여 Scratch File을 생성합니다다.

프로젝트 뷰를 Android로 설정하면 Scratch File를 확인할 수 없습니다. 프로젝트 뷰를 Project로 변경합시다.

이제 코드를 작성하고 실행 버튼을 누르면 왼쪽에서 결과를 확인할 수 있습니다.

또한 특정 모듈을 선택하여 모듈에 정의된 컴포넌트를 사용할 수도 있습니다.
