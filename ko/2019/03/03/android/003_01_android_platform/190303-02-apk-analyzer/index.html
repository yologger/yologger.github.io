[안드로이드] APK 분석하기 (APK Analyzer)

2019/03/03 카테고리 Android, Android Framework

안드로이드(Android)에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

관련 포스트

  • 앱에서 APK 추출하기 (APK Extractor)
  • APK 분석하기 (APK Analyzer) ⬅️ ⬅️ ⬅️
  • APK 디컴파일 하기 (jadx)

들어가며

안드로이드 스튜디오의 APK Analyzer를 사용하면 APK 파일을 분석할 수 있습니다.

APK Analyzer

Android Studio > Build > Analyze APK를 선택합니다.


APK 파일을 선택하면 분석 정보를 확인할 수 있습니다.

마치며

이번 포스트에서는 APK 파일을 분석하는 방법에 대해 알아봤습니다. 다음 포스트에서는 디컴파일에 대해 알아보겠습니다.
