[Kotlin] null 처리 - (1) NotNull과 Nullable

2019/01/19 카테고리 Kotlin

Kotlin의 특징과 개념에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

앞으로 4개의 포스트를 통해 Kotlin의 null 처리 방법에 대해 알아보려고 합니다.

  1. NotNull과 Nullable ⬅️ ⬅️ ⬅️
  2. Null Check
  3. 엘비스 연산자(Elvis Operator)
  4. 스코프 함수를 통한 Null Check

Not null 타입

Kotlin은 변수를 초기화하지 않으면 컴파일 자체가 안됩니다.

클래스의 멤버변수 역시 초기화하지 않으면 컴파일 자체가 안됩니다.

Not null 타입의 변수에는 null을 할당할 수도 없습니다.

Nullable 타입

Kotlin에서 null을 할당하기 위해서는 변수를 Nullable타입으로 선언해야합니다. 이때는 타입 뒤에 물음표(?)를 붙입니다.

물론 Nullable타입이라도 초기화하지 않으면 컴파일 자체가 되지 않습니다.
