[Kotlin] 클래스의 생성자

2019/01/07 카테고리 Kotlin

Kotlin의 특징과 개념에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

이전 포스트에는 객체지향 프로그래밍, 클래스, 인스턴스, 객체에 대해 알아봤습니다. 이번 포스트에서는 클래스의 생성자(Constructor)에 대해 알아보겠습니다.

생성자(Constructor)란?

생성자는 클래스의 인스턴스를 생성할 때 호출되는 구문입니다. 보통 생성자에서는 초기화 작업을 수행합니다. 이때 키워드constructor를 사용합니다.

이 초기화 구문은 클래스의 인스턴스를 생성할 때 호출됩니다.

생성자에는 2가지 종류가 있습니다.

1. 기본 생성자

기본 생성자는 클래스 이름 뒤에 constructor키워드를 붙여서 만듭니다. 주의할 점은 기본 생성자는 별도의 초기화 구문이 존재하지 않습니다. 이때는 init{ .. } 구문을 사용할 수 있습니다.

constructor키워드는 생략 가능합니다.

맴버변수 선언도 생성자 안에서 할 수 있습니다.

위 처럼 코드를 작성하는 경우, 클래스를 생성할 때 전달한 값 ronaldo가 멤버변수 name에 자동으로 초기화됩니다.

2. 보조 생성자

기본 생성자를 정의하지 않고 아래와 같이 보조 생성자만을 사용할 수 있습니다.
