변수

변수는 변수명 앞에 자료형을 붙이면 됩니다.

상수

상수는 변수명 앞에 자료형 앞에 final키워드를 붙이면 됩니다. 보통 상수의 이름은 대문자와 언더바(_)를 사용합니다. 변수는 값을 할당하고 변경할 수 있지만 상수는 값을 한번 할당하면 변경할 수 없습니다.
