[Java] instanceOf 란?

2018/01/24 카테고리 Java

Java의 특징과 개념에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

이전 포스트에서는 Java의 비교문에 대해 알아봤습니다. 이번 포스트에서는 키워드instanceOf에 대해 알아보겠습니다.

instanceOf

키워드instanceOf는 인스턴스가 특정 클래스의 타입인지 확인할 때 사용합니다.

마치며
