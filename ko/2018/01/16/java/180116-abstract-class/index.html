[Java] 추상 클래스(Abstract Class)

2018/01/16 카테고리 Java

Java의 특징과 개념에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

이전 포스트에서는 Java의 키워드final에 대해 알아봤습니다. 이번 포스트에서는 Java의 클래스의 추상 클래스(Abstract Class)에 대해 알아보겠습니다.

추상 클래스(Abstract Class)

미구현된 메소드를 가지고 있는 클래스를 추상 클래스(Abstract Class)라고 합니다. 추상 클래스를 선언할 땐 키워드 abstract를 사용합니다.

추상 클래스는 인스턴스를 생성할 수 없습니다.

따라서 이를 상속하는 자식 클래스에서 미구현된 메소드를 구현해야합니다.

이제 클래스의 인스턴스를 생성할 수 있습니다.

마치며

이번 포스트에서는 클래스의 추상 클래스(Abstract Class)에 대해 알아봤습니다. 다음 포스트에서는 인터페이스(Interface)에 대해 알아보겠습니다.
