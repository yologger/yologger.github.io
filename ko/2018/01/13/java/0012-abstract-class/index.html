[Java] 인터페이스 (interface)

2018/01/13 카테고리 Java

들어가며

인터페이스 (Interface)는 특정 규격을 준수하도록 강제하는데 사용합니다.

interface

인터페이스 (Interface)를 선언할 때는 키워드 interface를 사용합니다. 인터페이스에는 메서드를 선언만 하고 구현은 하지 않습니다.

클래스에서 인터페이스를 구현할 때는 키워드 implements를 사용합니다. 인터페이스 Database 규격을 따르는 클래스 2개를 구현해보겠습니다.

위 예제에서는 인터페이스 Database라는 규격을 따르도록 강제하고 있습니다. 이렇게 하면 Database를 사용하는 입장에서 실제 구현체가 무엇인지 신경쓰지 않고 사용할 수 있습니다. 이를 통해 유지보수성을 극대화할 수 있습니다.
