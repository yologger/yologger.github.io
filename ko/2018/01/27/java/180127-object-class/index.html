[자바] Object 클래스

2018/01/27 카테고리 Java

자바(Java)에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

이전 포스트에서는 접근 제한자(Access Modifier)에 대해 알아봤습니다. 이번 포스트에서는 Object 클래스에 대해 알아보겠습니다.

java.lang 패키지

java.lang 패키지는 자바에서 가장 기본적인 동작을 수행하는 클래스들의 집합입니다. java.lang 패키지에 있는 클래스는 import하지 않아도 바로 사용할 수 있습니다.

Object 클래스

Object클래스는 모든 자바 클래스의 최고 조상 클래스입니다. 자바의 모든 클래스는 암시적으로든 명시적으로든 Object클래스를 상속받고 있습니다. Object클래스는 java.lang패키지에 포함되며, Object클래스는 여러 메소드를 제공하고 있습니다.

(1) equals()

메소드를 호출하는 객체와 인자로 전달받은 객체가 동일한지 여부를 반환합니다. 다만 Object클래스를 상속받는 클래스에서는 equals()을 직접 구현해야합니다.

다음과 같이 인스턴스를 비교할 수 있습니다.

(2) getClass()

인스턴스의 클래스 타입을 반환합니다.

(3) hashcode()

hashcode()는 객체의 해시코드를 반환합니다. 해시코드는 객체의 주소값을 해시함수에 넣어 반환된 고유한 정수값입니다. 따라서 두 객체가 동일한 객체인지 비교할 때 사용할 수 있습니다.

(4) toString() 메소드

해당 객체의 정보를 문자열로 반환합니다.

(5) 그 외

그 외에 Object 클래스는 clone(), finalize(), notify(), notifyAll(), wait() 메소드도 제공합니다.

마치며
