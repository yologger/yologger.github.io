[자바] 익명 클래스(Anonymous Class)

2018/01/19 카테고리 Java

자바(Java)에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

이전 포스트에서는 내부 클래스(Inner Class)에 대해 알아봤습니다. 이번 포스트에서는 익명 클래스(Anonymous Class)에 대해 알아보겠습니다.

익명 클래스

부모 클래스를 상속할 때 자식 클래스를 한번만 사용하는 경우, 매번 자식 클래스를 정의하기에는 번거로운 경우가 있습니다.

이때 익명 클래스를 유용하게 사용할 수 있습니다.

익명 클래스는 안드로이드 어플리케이션에서 이벤트를 처리할 때 많이 사용합니다.

마치며

이번 포스트에서는 익명 클래스(Anonymous Class)에 대해 알아봤습니다. 다음 포스트에서는 열거형(enum)에 대해 알아보겠습니다.
