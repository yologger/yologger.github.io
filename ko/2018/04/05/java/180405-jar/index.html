JAR

JAR(Java Archive)파일은 자바 애플리캐이션을 효율적으로 배포하기 위해 만들어진 파일입니다. JAR파일은 여러 클래스 파일, 리소스 파일, 매니페스트 파일로 구성되어있습니다.
