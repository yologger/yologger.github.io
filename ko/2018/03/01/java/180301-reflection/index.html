[Java] 리플렉션(Reflection)

2018/03/01 카테고리 Java

Java의 특징과 개념에 대해 연재하고 있습니다. 목차는 이 곳에서 확인할 수 있습니다.

들어가며

리플렉션(Reflection)은 런타임에서 구체적인 클래스 타입을 몰라도 그 클래스의 멤버변수, 메소드 등에 접근할 수 있도록 하는 자바 API와 기법입니다. 리플렉션과 관련된 API는 java.lang.reflect.*패키지에 있습니다.

Class클래스와 클래스 참조

자바의 모든 클래스와 인터페이스는 컴파일 후 .class파일로 변환됩니다. .class파일에는 생성자, 멤버변수, 메서드 등의 객체 정보가 포함되어있는데 Class클래스는 이 .class파일에서 가져온 객체의 정보를 담는 역할을 합니다. Class클래스에 정보를 담는 방법에는 3가지가 있습니다.
