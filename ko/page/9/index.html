들어가며

이번 포스트에서는 키워드 final에 대해 알아보겠습니다.

1. 상수

키워드 final은 변수를 변경할 수 없는 상수로 만드는데 사용합니다.

2. 상속 불가능한 클래스

부모 클래스에 final을 붙이면 상속이 불가능합니다.

3. 오버라이드 불가능한 메소드

부모 클래스 메소드 앞에 final을 붙으면 오버라이드가 불가능합니다.

[Java] 변수와 상수

2018/01/06 카테고리 Java

들어가며

이번 포스트에서는 Java의 변수와 상수에 대해 알아보겠습니다.

변수

변수는 변수명 앞에 자료형을 붙이면 됩니다.

상수

상수는 변수명 앞에 자료형 앞에 final키워드를 붙이면 됩니다. 보통 상수의 이름은 대문자와 언더바(_)를 사용합니다. 변수는 값을 할당하고 변경할 수 있지만 상수는 값을 한번 할당하면 변경할 수 없습니다.
