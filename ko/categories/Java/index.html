[Java] 익명 클래스 (Anonymous Class)

2018/01/15 카테고리 Java

클래스를 상속하여 사용할 때 자식 클래스를 한번만 사용하는 경우, 매번 자식 클래스를 정의하기에는 번거로운 경우가 있습니다.

이때 익명 클래스(Anonymous Class)를 유용하게 사용할 수 있습니다.

익명 클래스는 안드로이드 어플리케이션에서 이벤트를 처리할 때 많이 사용합니다.
