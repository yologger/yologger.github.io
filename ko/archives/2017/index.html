[Java] Oracle JDK vs. OpenJDK

2017/01/04 카테고리 Java

Oracle JDK

사용 목적에 따라 유료이기도 무료이기도 한 JDK입니다. Mac OS에는 JDK가 기본적으로 설치되어 있습니다.

OpenJDK

사용 목적에 관계없이 무료인 JDK입니다. 안드로이드 스튜디오에는 OpenJDK가 내장되어 있으므로 별도의 JDK 설치가 필요없습니다. File > Project Structure > SDK Location에서 경로를 확인할 수 있습니다.

보통 compileSDKVersion에 따라 적당한 JDK 버전이 설정됩니다. 물론 직접 JDK 버전을 설정할 수 있습니다.

[Java] Java Platform(JDK) 버전과 종류

2017/01/03 카테고리 Java

들어가며

Java Platform(JDK)에는 여러 에디션이 존재합니다. 이번 포스트에서는 대표적인 3가지 에디션의 특징에 대해 알아보겠습니다.

Java SE (Java Standard Edition)

  1. 과거에는 J2SE(Java 2 SE)로 불렸다.
  2. 가장 보편적으로 사용되는 에디션입니다.
  3. 자바 데스크탑, 모바일 어플리케이션 개발에 사용됩니다.
  4. JDK 외에 Swing, AWT 등 데스크탑 개발에 필요한 API가 포함됩니다.

Java EE (Java Enterprise Edition)

  1. 과거에는 J2EE(Java 2 EE)로 불렸습니다.
  2. 자바 서버 애플리케이션 에 사용된다.
  3. Java SE의 모든 API도 포함된다.
  4. 그 외 EJB, JSP, Sevlet, JDBC등 서버 어플리케이션 개발을 위한 API가 포함된다.

Java ME (Java Micro Edition)

  1. Java SE API의 일부만 포함
  2. 셋톱박스, PDA등 임베디드 시스템을 개발하는데 사용됩니다.

[Java] JVM, JRE, JDK의 차이

2017/01/01 카테고리 Java

들어가며

이번 포스트에서는 JVM, JRE, JDK의 차이점에 대해 알아보겠습니다.

JVM

JVM은 Java Virtual Machine의 줄임말로 .class파일을 운영체제에 종속적인 코드로 변환해주는 역할을 합니다.

JRE

JRE는 Java Runtime Edition의 줄임말로 Java언어로 작성된 애플리케이션을 실행하는데 필요한 요소들입니다. JVM과 실행에 필요한 Java API로 구성되어 있습니다.

JDK

JDK는 Java Development Kit의 줄임말로 어플리케이션 개발에 필요한 요소들입니다. JRE과 Java Compiler, Java Debugger 등 개발에 필요한 다양한 도구를 포함하고 있습니다. 보통 Java Platform은 JDK를 의미합니다.
