[Kotlin] 상수 (val, const)

2019/01/28 카테고리 Kotlin

var

변수를 선언할 때는 키워드 var를 사용합니다. 변수는 런타임에서 값을 수시로 변경할 수 있습니다.

val

val은 런타임에 값이 결정되는 상수입니다. 상수는 한 번 값이 할당되면 변경할 수 없습니다.

val은 문자열도 할당할 수 있습니다.

val는 클래스의 인스턴스도 할당할 수 있습니다.

const val

const val는 컴파일 타임에 값이 결정되는 상수입니다. 인스턴스를 할당할 수 없으며, 기본 자료형과 문자열만 할당할 수 있습니다.

클래스 안에서 사용할 때는 companion object블럭 안에서 정의합니다.
