[Kotlin] 집합 자료형 (Collection) - Map

2019/01/04 카테고리 Kotlin

들어가며

이전 포스트에서는 집합자료형 (Collection)에서 List와 Set에 대해 알아봤습니다. 이번 포스트에서는 Map에 대해 알아보겠습니다.

1. List

변경 불가능한 List를 선언할 때는 listOf()메소드를 사용합니다.

변경 가능한 List를 선언할 때는 mutableListOf()메소드를 사용합니다.

2. Set

변경 불가능한 Set를 선언할 때는 setOf()메소드를 사용합니다.

변경 가능한 Set를 선언할 때는 mutableSetOf()메소드를 사용합니다.

3. Map

변경 불가능한 Map을 선언할 때는 mapOf()메소드를 사용합니다.

변경 가능한 Map를 선언할 때는 mutableMapOf()메소드를 사용합니다.

4. 집합자료형 조작 함수

집합자료형은 데이터 조작을 위한 다양한 메소드를 제공합니다. 대표적인 메소드 몇 가지만 알아보겠습니다.

[Kotlin] 집합 자료형 (Collection) - List

2019/01/04 카테고리 Kotlin

집합자료형 (Collection)

집합자료형 (Collection)을 사용하면 같은 타입의 데이터를 여러 개 담을 수 있습니다. Kotlin은 List, Set, Map이라는 3개의 집합자료형을 지원합니다. List, Set, Map은 모두 Collection클래스를 상속받습니다.

List

변경 불가능한 List를 선언할 때는 listOf()메소드를 사용합니다.

변경 가능한 List를 선언할 때는 mutableListOf()메소드를 사용합니다.

정리

List는 Array와 매우 유사합니다. 그러나 바로 위 예제 코드처럼 List는 초기화 후 값을 추가, 삭제할 수 있습니다. 반면 Array는 초기화하여 크기가 결정되면 값을 추가, 삭제할 수 없습니다.
