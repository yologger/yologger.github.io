[Java] JVM, JRE, JDK 차이

2018/01/01 카테고리 Java

들어가며

이번 포스트에서는 JVM, JRE, JDK의 차이점에 대해 알아보겠습니다.

JVM

JVM은 Java Virtual Machine의 줄임말로 자바 바이트 코드(.class 파일)를 운영체제에 종속적인 코드로 변환하여 실제로 실행하는 역할을 합니다.

JRE

JRE는 Java Runtime Edition의 줄임말로 Java언어로 작성된 애플리케이션을 실행하는데 필요한 요소들입니다. JVM과 실행에 필요한 Java API로 구성되어 있습니다.

JDK

JDK는 Java Development Kit의 줄임말로 어플리케이션 개발에 필요한 요소들입니다. JRE외에도 Java Compiler, Java Debugger 등 개발에 필요한 다양한 도구를 포함하고 있습니다. 보통 Java Platform은 JDK를 의미합니다.
